# Threaded Bank Transaction System (Python)

A simple Python project demonstrating **multithreading** and **thread synchronization** using the `threading` module. The system processes deposits and withdrawals concurrently while preventing race conditions with `Lock`.

## Features

* Concurrent deposits and withdrawals
* Thread-safe account balance using `threading.Lock()`
* Prevents race conditions
* Displays transaction history and final balance

## Technologies Used

* Python 3
* `threading`
* `time`

## How to Run

```bash
python bank_transaction.py
```
